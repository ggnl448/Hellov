# AI Поиск — Android (Kotlin)

Нативное Android-приложение, повторяющее твой веб-прототип (Perplexity-style поиск),
но AI-ответы и поисковые системы грузятся в настоящих `WebView` внутри приложения,
а не открываются в браузере.

## Что реализовано

- **Вкладки браузера** (как в Chrome) — можно открывать несколько вкладок кнопкой «+»,
  закрывать крестиком, переключаться. Каждая вкладка хранит свои 3 WebView (по одному
  на под-вкладку), так что состояние не теряется при переключении.
- **Под-вкладки внутри каждой вкладки**: «Ответ» / «Ссылки» / «Изображения» — как в
  твоём веб-варианте.
- **«Ответ»**: переключатель между Gemini, ChatGPT.com и Claude.ai (кнопка
  «сменить ИИ» циклически переключает, либо можно нажать конкретный чип).
- **«Ссылки» / «Изображения»**: переключатель между Google, Bing, DuckDuckGo, Yandex
  (выпадающее меню по нажатию на текущее название + ссылка «сменить поиск» для
  цикличного переключения).
- Ссылки, открывающиеся в новом окне (`target="_blank"`), автоматически открываются в
  новой вкладке приложения.
- Системная кнопка «Назад» сначала листает историю текущего WebView, а не сразу
  закрывает приложение.

## Структура проекта

```
app/src/main/java/com/maxdev/aisearchbrowser/
  Models.kt        — enum SearchEngine, AiService, SubTab + построение URL
  BrowserTab.kt     — состояние одной вкладки (3 WebView + текущие настройки)
  MainActivity.kt   — вся UI-логика (интерфейс собран программно, без XML-layout)
```
.

## Товарные знаки

Приложение обращается к публичным сайтам Google Gemini, OpenAI ChatGPT и Anthropic
Claude по их обычным URL — это не то же самое, что использование чужого кода или
логотипов. Как и с твоим веб-прототипом: не используй официальные логотипы, 
Проект не аффилирован и не одобрен OpenAI, Mozilla, duckduckgo, Microsoft Яндекс,Google или Anthropic. Названия сервисов используются исключительно для указания внешних ссылок.
